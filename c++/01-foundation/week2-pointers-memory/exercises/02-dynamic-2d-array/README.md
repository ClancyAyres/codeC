# 习题 02：动态二维数组

## 目标

使用 `new` 动态分配二维数组，填充乘法表，用指针算术遍历，最后正确释放。

## 要求

1. 从标准输入读取行数 m 和列数 n
2. 使用 `new` 分配 m×n 的二维 int 数组（先分配行指针数组，再为每行分配列数组）
3. 填充 `arr[i][j] = (i + 1) * (j + 1)`（乘法表）
4. 使用纯指针算术 `*(*(arr + i) + j)` 打印整个矩阵
5. 使用 `delete[]` 正确释放所有内存（先释放每行，再释放行指针数组）

## 提示

```cpp
// 分配
int** arr = new int*[m];
for (size_t i = 0; i < m; ++i) {
    arr[i] = new int[n];
}

// 释放（与分配顺序相反）
for (size_t i = 0; i < m; ++i) {
    delete[] arr[i];
}
delete[] arr;
```

## 示例输入输出

```
输入: m=3, n=4
输出:
  1  2  3  4
  2  4  6  8
  3  6  9 12
```
